About NewsNER

NewsNER is a human-in-the-loop Named Entity Recognition pipeline designed for extracting structured information from text.

What It Does

  • 🤖 Automatic Extraction — Uses spaCy transformer models to identify entities
  • 📊 Confidence Scoring — Every entity gets a confidence score (0-100%)
  • 👤 Human Review — Low-confidence predictions flagged for expert validation
  • 📈 Quality Metrics — Measures agreement between model and human annotations (Cohen's Kappa)

Entity Types

ORG
Organizations & Companies
PERSON
People & Names
GPE
Locations & Places
MONEY
Monetary Values
DATE
Dates & Times
EVENT
Events & Occurrences

Use Cases

  • 📰 Financial news curation and market intelligence
  • 🔬 Biological literature mining and biomedical data extraction
  • 📊 Research data collection and knowledge base construction
  • 🏢 Document classification and information extraction

Tech Stack

Backend: Python 3.9+, Flask, spaCy 3.7+
Frontend: HTML5, CSS3, Vanilla JavaScript
NLP: Transformer-based models, confidence estimation
Quality: Cohen's Kappa agreement metrics

Open Source

NewsNER is built for research and production use. Check out the GitHub repository for the full source code.

How to Use

Step 1: Enter Text

Paste any text into the textarea. Works with:

  • 📰 News articles
  • 💼 Business reports
  • 📄 Research papers
  • 💬 Any English text

Step 2: Extract Entities

Click "Extract Entities" button or press Ctrl+Enter

The AI model will analyze your text and identify all named entities in real-time.

Step 3: Review Results

See:

  • Statistics — Total entities found, high confidence count
  • Entity List — Each entity with confidence score
  • Color Coding — Entity types color-coded for quick scanning

Understanding Confidence

  • 🟢 90-100% — High confidence, likely correct
  • 🟡 75-90% — Medium confidence, should verify
  • 🔴 <75% — Low confidence, flagged for review

Tips

  • ✨ Try the example buttons to see how it works
  • 📝 Longer texts (200+ words) work best
  • 🔄 Complex entity relationships are captured
  • ⚡ Results appear instantly

Advanced: Configure Entity Types

To customize which entities to extract, edit config.yaml:

ner:
  entity_types:
    - ORG
    - PERSON
    - MONEY
    # Add or remove types here
